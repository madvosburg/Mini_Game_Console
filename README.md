# Mini_Game_Console

EE 333 Electronic Systems Design:

The miniature game console is a compact, battery-powered handheld device designed to play classic games such as Snake and Pong. The system uses an ESP32 microcontroller for game logic, input processing, and peripheral control. Games are stored on an SD card and can be loaded dynamically, allowing flexibility in game selection.
